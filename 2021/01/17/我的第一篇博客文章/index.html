我的第一篇博客文章

##第一章 新建一个md文件

  1. 通过管理员进入命令控制符
  2. cd blog/ 进入blog文件夹下然后输入
    hexo n “文章标题”
  3. cd source/_posts/进入到存放md文件的文件夹内然后输入
    dir 查看是否创建成功

##第二章 编辑新建的文件

进入C:\Windows\System32\blog\source\ _posts内用vscode打开编辑即可

##第三章 错误应对
输入 cnpm install
